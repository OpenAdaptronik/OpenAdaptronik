Beispiel für Batch-Ausführung von SciLab
 - SciLab muss installiert sein und der Pfad in der Batch-Datei muss auf die installierte Version von SciLab zeigen.
 - der Aufruf erfolgt mit:
>execute_script.bat <m> <k> <d>
Die Ergebnisse werden in eigf.txt geschrieben/überschrieben (w0 und wd in rad/s)
Ein plot der Transferfunktion (abs und arg, Bode-artig) wird in funplot.png gespeichert
